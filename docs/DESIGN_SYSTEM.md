# Design System

## Dual-Theme Architecture
Squareshare.to employs a stark, dual-theme strategy to guide the user's emotional journey.

### Theme 1: Premium Dark
- **Background**: Stark black (`#000000`).
- **Texture**: Blurred, high-opacity noisy mesh gradients pushed deep into the background z-index to create depth without distraction.
- **Accent**: Acid Green (`#CCFF00`) used aggressively but sparingly—only for primary Call-To-Action (CTA) elements to guarantee focus.
- **Typography**: High-contrast white/light-gray text.

### Theme 2: Clean White
- **Background**: Brutally clean white/light gray (`#FFFFFF` or `#F9F9F9`).
- **Role**: Acts as a minimalist wrapper for product showcases, allowing the content to take center stage without visual noise.

## Layout System: The Bento Grid
- **Structure**: Asymmetric "Bento Grid" component system.
- **Block Sizes**: Utilizes standard modular ratios (1x1, 2x2, 2x1 blocks).
- **Aesthetics**: Sharp, high-contrast borders with standard `16px` (`rounded-2xl`) corners to maintain a premium feel.
- **Philosophy**: UX is always prioritized over UI. The grid must be legible, easily scannable, and logically structured before aesthetic flourishes are applied.
